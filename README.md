# App de Sorteio de Chaveamento - Tênis

App web simples para sortear confrontos de mata-mata e gerar o chaveamento completo.

## Como usar

1. Abra `index.html` no navegador.
2. Insira um participante por linha.
3. Clique em **Sortear Chaveamento**.
4. Clique em um nome para avançar o vencedor de cada jogo.

## Observações

- O sistema cria byes automaticamente quando o número de participantes não é potência de 2.
- Para 12 participantes, são criados 4 byes no primeiro round.
